Keyboard Event Logger (Learning Project)

This is a simple keyboard event logger built using Python.
It logs key presses only when the user starts logging manually.

Purpose:
- Learn keyboard event handling
- Learn GUI programming with Tkinter
- Understand system-level input listeners

Features:
- Start and stop logging manually
- Displays pressed keys in a window
- Handles special keys like Enter and Space
- Visible and ethical design

Technologies Used:
- Python
- Tkinter
- pynput

How to Run:
1. Install dependency:
   pip install pynput

2. Run the program:
   python key_logger.py

Ethical Notice:
This project is for learning purposes only.
It does not log keys secretly or run in the background.
